# Documentation Technique MaxOS

## Introduction

Bienvenue dans la documentation technique de MaxOS, un système d'exploitation minimaliste et expérimental conçu pour l'architecture x86. MaxOS est un projet "bare metal", ce qui signifie qu'il interagit directement avec le matériel sans dépendre d'un système d'exploitation hôte. Ce projet sert de plateforme d'apprentissage et d'expérimentation pour comprendre les rouages fondamentaux d'un système d'exploitation, de son démarrage à la gestion des ressources de base.
