# Documentation Technique MaxOS

## 1. Vue d'ensemble de MaxOS

MaxOS est un système d'exploitation minimaliste en cours de développement, conçu comme un projet éducatif et une plateforme d'expérimentation pour l'apprentissage des concepts fondamentaux des systèmes d'exploitation. Actuellement à l'état de prototype "bare metal" (score 35/100), il est capable de démarrer sur une architecture x86 et d'afficher du texte via le mode VGA 8
